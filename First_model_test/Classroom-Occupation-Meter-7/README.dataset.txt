# Dataset for Howest CTAI Project 1


## Description

This dataset is a combination of other datasets found on Roboflow and Kaggle.

It consists of pictures of people, with bounding boxes drawn around their heads.


### Datasets used:

1. https://www.kaggle.com/datasets/trainingdatapro/people-tracking
2. https://www.kaggle.com/datasets/constantinwerner/human-detection-dataset
3. https://universe.roboflow.com/ricky-sambora/crowd-detection-7suou
4. https://universe.roboflow.com/arizona-state-university-kgkyn/people-detection-jhhbd/dataset/1
